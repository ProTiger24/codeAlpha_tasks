# 🤖 AI Chatbot — CodeAlpha Internship Task 3

An intelligent chatbot application built with Java backend and modern web frontend,
powered by Groq's LLaMA AI model.

 🛠 Tech Stack
- **Backend:** Java 21, Spark Framework
- **AI Model:** LLaMA 3.3 70B (via Groq API)
- **Frontend:** HTML, CSS, JavaScript
- **Build Tool:** Maven

✨ Features
- Real-time AI-powered responses
- Bengali & English language support
- Clean and responsive chat UI
- RESTful API backend

🚀 How to Run
1. Clone the repository
2. Add your Groq API key in `ClaudeService.java`
3. Run: `mvn clean install && mvn exec:java`
4. Open: `http://localhost:8080`
